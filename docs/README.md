# RelishEconomy

Advanced multi-currency economy plugin for Paper 1.21+.

## Key Features

### Free
- Multiple currencies (symbols, colors, starting balances, per-currency permissions)
- Vault economy provider (compatible with Vault-dependent plugins)
- PlaceholderAPI placeholders (balances, formatted balances, baltop, ranks, stats)
- Player payments (`/pay`) and leaderboards (`/baltop`)
- Currency exchange (`/exchange`) with per-currency exchange rates
- Config updater/validator (restores missing bundled files, merges missing keys, creates timestamped backups)

### Premium
- Shop GUI (categories, paging, searching)
- Purchase GUI flow (quantity selector, confirm/cancel/back to same page)
- Favorites (favorites view, `favorites.yml`, optional glow display)
- Optional "show unpriced items" (disabled items shown instead of hidden)
- Sell GUI + composter selling
- Physical currency (`/withdraw`) with per-currency customization:
  - template material/name/lore
  - custom model data
  - optional crafting recipes
